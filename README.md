# Pong Game (HTML5 & JavaScript)

A classic **Pong game** built using **HTML5 Canvas** and **vanilla JavaScript**.  
The game runs entirely in a **web browser** and requires no external libraries.

---

## Features

- Classic Pong gameplay
- Player vs Computer
- Mouse-controlled paddle
- Ball collision physics
- Score tracking
- Win screen with restart option
- Center net for visual polish

---

## Technologies Used

- **HTML5**
  - `<canvas>` element for rendering the game
- **JavaScript**
  - Game loop logic
  - Collision detection
  - Player input handling
  - Computer-controlled paddle (AI)

No frameworks or libraries were used — everything is written in plain JavaScript.

---

## How to Play

- Move your **mouse up and down** to control the **left paddle**
- The **right paddle** is controlled by the computer
- First player to reach **3 points** wins
- Click the mouse to restart after a win
